## [v0.1.0] - 2026-02-23

### Highlights

- Go framework for analyzing compliance regulations with machine-readable data structures
- CLI tool for loading, querying, and validating compliance frameworks
- Web viewer for browsing compliance data (GitHub Pages compatible)

### Added

- Core compliance data model: Jurisdiction, Regulation, Requirement, Solution, Mapping, ZoneAssignment, EnforcementAssessment types ([`ff5ba52`](https://github.com/grokify/go-comply/commit/ff5ba52))
- ComplianceFramework container with query methods for requirements, solutions, mappings, and zones ([`ff5ba52`](https://github.com/grokify/go-comply/commit/ff5ba52))
- ComplianceAnalysis and ExecutiveOverview types for strategic reporting ([`ff5ba52`](https://github.com/grokify/go-comply/commit/ff5ba52))
- ResearchInput processing with validation and analysis ([`ff5ba52`](https://github.com/grokify/go-comply/commit/ff5ba52))
- JSON file I/O utilities for loading and saving compliance frameworks ([`ff5ba52`](https://github.com/grokify/go-comply/commit/ff5ba52))
- `comply` CLI with load, list, query, validate, coverage, and import-research commands ([`90f6ee7`](https://github.com/grokify/go-comply/commit/90f6ee7))
